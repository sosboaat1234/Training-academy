# วิธีทำให้ทุกคนเปิดเว็บได้ จากที่ไหนก็ได้ (ฟรี ไม่ต้องเช่า Server / Domain)

ไฟล์ `index.html` เป็นไฟล์เดียวจบ เปิดได้ 2 ระดับ:

## ระดับ 1 — เปิดบนเครื่องตัวเอง (ทดสอบ)
ดับเบิลคลิกไฟล์ `index.html` ได้เลย เปิดในเบราว์เซอร์ทันที ใช้ดูคนเดียว/ทดสอบ
แต่คนอื่นจะเปิดไม่ได้ เพราะไฟล์อยู่ในเครื่องเรา

## ระดับ 2 — ให้ทุกคนเข้าได้จากที่ไหนก็ได้ (ฟรี 100%)
เลือกวิธีใดวิธีหนึ่งด้านล่าง ทุกวิธีฟรี ได้ลิงก์ https ใช้แชร์ได้เลย

### ตัวเลือก A: GitHub Pages (แนะนำ — ฟรีถาวร)
1. สมัคร GitHub ที่ https://github.com (ฟรี)
2. กด **New repository** ตั้งชื่อเช่น `training-academy` แล้วกด Create
3. กด **Add file > Upload files** แล้วลากไฟล์ `index.html` ขึ้นไป กด Commit
4. ไปที่ **Settings > Pages** > ช่อง Branch เลือก `main` แล้วกด Save
5. รอ 1–2 นาที จะได้ลิงก์แบบ `https://ชื่อบัญชี.github.io/training-academy/`
   ส่งลิงก์นี้ให้ใครก็ได้ เปิดได้ทั้งคอมและมือถือ

### ตัวเลือก B: Netlify Drop (ง่ายที่สุด — ลากวาง)
1. เข้า https://app.netlify.com/drop
2. ลากไฟล์ `index.html` (หรือทั้งโฟลเดอร์) ไปวางบนหน้าเว็บ
3. ได้ลิงก์ทันที เช่น `https://ชื่อสุ่ม.netlify.app` (เปลี่ยนชื่อได้ภายหลัง)

### ตัวเลือก C: Cloudflare Pages
1. เข้า https://pages.cloudflare.com สมัครฟรี
2. สร้างโปรเจกต์ใหม่ อัปโหลดไฟล์ ได้ลิงก์ `.pages.dev`

## สำคัญ: ต้องอัปโหลด 2 ไฟล์คู่กันเสมอ
เว็บนี้ใช้โลโก้ SITEM จากไฟล์ `sitem-mark.png` ดังนั้นเวลาอัปโหลด/แชร์ ต้องเอาไปทั้ง 2 ไฟล์:
`index.html` **และ** `sitem-mark.png` (วางไว้โฟลเดอร์เดียวกัน) ไม่งั้นโลโก้จะไม่ขึ้น

## หมายเหตุ
- เว็บนี้ใช้ฟอนต์ IBM Plex ผ่าน CDN จึงต้องต่อ **อินเทอร์เน็ต** ขณะเปิด (ปกติทุกคนมีอยู่แล้ว)
- อยากได้ชื่อโดเมนสวยๆ ของตัวเอง (เช่น `training.company.com`) ค่อยซื้อโดเมนทีหลังได้ แต่ **ไม่จำเป็น** ลิงก์ฟรีด้านบนใช้งานได้เต็มที่
- แก้เนื้อหา/วิดีโอ/คำถาม: เปิด `index.html` หาส่วน `const PRODUCTS = [...]` ด้านบนของ `<script>` แก้ได้เลย แล้วอัปโหลดทับไฟล์เดิม
