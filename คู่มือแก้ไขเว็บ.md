# คู่มือแก้ไขเว็บ SITEM Academy ด้วยตัวเอง

แก้ทุกอย่างได้ในไฟล์ `index.html` ไฟล์เดียว เปิดด้วยโปรแกรมแก้ข้อความอะไรก็ได้
(Notepad, VS Code, Notepad++) แล้วบันทึก จากนั้นอัปโหลดทับไฟล์เดิม

> จุดเดียวที่ต้องแก้คือก้อนข้อมูล **`const PRODUCTS = [ ... ]`**
> (อยู่ใต้คอมเมนต์ `1) DATA` ในส่วน `<script>`)
> ไม่ต้องแตะส่วน CSS หรือฟังก์ชันอื่นเลย ระบบจะอัปเดตเมนู/วงแหวน %/ค้นหา ให้เองอัตโนมัติ

---

## โครงสร้างของ 1 ผลิตภัณฑ์

แต่ละผลิตภัณฑ์คือ 1 บล็อกในวงเล็บปีกกา `{ ... }` คั่นด้วยจุลภาค `,` หน้าตาแบบนี้:

```js
{
  id:"power",                          // รหัสเฉพาะ ห้ามซ้ำกับตัวอื่น (ภาษาอังกฤษ ไม่มีเว้นวรรค)
  icon:"zap",                          // ไอคอน (เลือกจากรายการด้านล่าง)
  color:"var(--amber-500)",            // สีพื้นไอคอน (เลือกจากรายการด้านล่าง)
  eyebrow:"POWER",                     // ป้ายหมวดตัวพิมพ์ใหญ่เหนือชื่อ
  name:"ระบบสำรองไฟ UPS",              // ชื่อผลิตภัณฑ์ (โชว์ในเมนูและหัวเรื่อง)
  tagline:"Static UPS — Riello, Eaton, Piller",   // คำโปรยสั้นๆ
  description:"ข้อความอธิบายผลิตภัณฑ์ ใส่ยาวได้",
  howto:[                              // วิธีใช้งานเป็นข้อๆ ใส่กี่ข้อก็ได้
    "ขั้นตอนที่ 1 ...",
    "ขั้นตอนที่ 2 ...",
  ],
  videoId:"aqz-KE-bpKQ",              // รหัสวิดีโอ YouTube (ดูวิธีหาด้านล่าง)
  quiz:[                              // แบบทดสอบ ใส่กี่ข้อก็ได้
    { q:"คำถามข้อ 1?", options:["ก","ข","ค","ง"], correct:1 },
    { q:"คำถามข้อ 2?", options:["ก","ข","ค","ง"], correct:0 },
  ],
},
```

**กฎสำคัญ 3 ข้อ (กันเว็บพัง):**
1. ทุกบรรทัดในวงเล็บต้องลงท้ายด้วยจุลภาค `,`
2. ข้อความต้องอยู่ในเครื่องหมายคำพูด `"..."` เสมอ — ถ้าในข้อความมี `"` ให้เปลี่ยนเป็น `'` แทน
3. `correct` คือ "ลำดับของคำตอบที่ถูก" โดย **เริ่มนับจาก 0** → ตัวเลือกแรก=0, ที่สอง=1, ที่สาม=2, ที่สี่=3

## ➕ วิธีเพิ่มผลิตภัณฑ์ใหม่

ก๊อปบล็อกข้างล่างนี้ไปวาง **ก่อนบรรทัด `];`** ที่ปิดท้าย `PRODUCTS` แล้วแก้เนื้อหา:

```js
{
  id:"newproduct",
  icon:"box",
  color:"var(--blue-500)",
  eyebrow:"หมวดของฉัน",
  name:"ชื่อผลิตภัณฑ์ใหม่",
  tagline:"คำโปรยสั้นๆ",
  description:"รายละเอียดผลิตภัณฑ์...",
  howto:[
    "ขั้นตอนที่ 1",
    "ขั้นตอนที่ 2",
    "ขั้นตอนที่ 3",
  ],
  videoId:"ใส่รหัสวิดีโอตรงนี้",
  quiz:[
    { q:"คำถามข้อ 1?", options:["ตัวเลือก 1","ตัวเลือก 2","ตัวเลือก 3","ตัวเลือก 4"], correct:0 },
    { q:"คำถามข้อ 2?", options:["ตัวเลือก 1","ตัวเลือก 2","ตัวเลือก 3","ตัวเลือก 4"], correct:2 },
    { q:"คำถามข้อ 3?", options:["ตัวเลือก 1","ตัวเลือก 2","ตัวเลือก 3","ตัวเลือก 4"], correct:1 },
  ],
},
```

เซฟแล้วเปิดเว็บ จะมีเมนูใหม่ขึ้นเองในแถบข้าง และวงแหวน % รวมจะนับเพิ่มให้อัตโนมัติ

## ➖ วิธีลบผลิตภัณฑ์

หาบล็อก `{ ... }` ของผลิตภัณฑ์ที่ไม่ต้องการ แล้วลบทั้งบล็อก
(ตั้งแต่ `{` จนถึง `},` ที่ปิด) ออกให้หมด เซฟ เสร็จ

---

## ✏️ วิธีแก้/เพิ่ม/ลด ข้อสอบ

ไปที่ `quiz:[ ... ]` ของผลิตภัณฑ์นั้น แต่ละข้อคือ 1 บรรทัดแบบนี้:

```js
{ q:"คำถาม?", options:["A","B","C","D"], correct:2 },
```

- **เพิ่มข้อ:** ก๊อป 1 บรรทัดมาวางเพิ่ม
- **ลดข้อ:** ลบบรรทัดนั้นทิ้ง
- **เปลี่ยนจำนวนตัวเลือก:** เพิ่ม/ลดสมาชิกใน `options` (จะกี่ตัวก็ได้ ไม่จำเป็นต้อง 4)
- **เกณฑ์ผ่าน:** ระบบตั้งไว้ที่ 70% ของจำนวนข้อโดยอัตโนมัติ เช่น 3 ข้อต้องได้ 3, 5 ข้อต้องได้ 4

> ตัวอย่าง: ถ้าคำตอบที่ถูกคือตัวเลือกที่ 3 (`"C"`) ให้ใส่ `correct:2`

## 🎬 วิธีใส่วิดีโอ (รองรับทั้ง YouTube และ Google Drive)

ช่อง `videoId` วางได้หลายแบบ ระบบจะแปลงให้เอง:

**YouTube** — วางได้ทั้ง 3 แบบ
```js
videoId:"aqz-KE-bpKQ",                                  // รหัสล้วน
videoId:"https://www.youtube.com/watch?v=aqz-KE-bpKQ",  // ลิงก์เต็ม
videoId:"https://youtu.be/aqz-KE-bpKQ",                 // ลิงก์สั้น
```

**Google Drive** — วางลิงก์แชร์ของไฟล์วิดีโอได้เลย
```js
videoId:"https://drive.google.com/file/d/1AbC_dEfGhIjK/view?usp=sharing",
```

> ⚠️ **สำคัญสำหรับ Drive:** ต้องตั้งค่าแชร์ไฟล์เป็น **"ทุกคนที่มีลิงก์ (Anyone with the link)"** ก่อน
> ไม่งั้นคนอื่นจะเปิดวิดีโอไม่ได้ (ขึ้นจอดำ/ขอสิทธิ์เข้าถึง)
> วิธีตั้ง: คลิกขวาที่ไฟล์ใน Drive > แชร์ > เปลี่ยน "ผู้ที่เข้าถึงได้แบบจำกัด" เป็น "ทุกคนที่มีลิงก์" > เสร็จ

## 🎨 ตารางไอคอนและสีที่ใช้ได้

**ไอคอน (`icon`)** — ใส่ได้ตามนี้:
`building` (อาคาร/บริษัท), `zap` (ไฟ/พลังงาน), `chart` (กราฟ/มอนิเตอร์),
`snow` (ความเย็น), `droplet` (น้ำ/ดับเพลิง), `box` (กล่อง/ทั่วไป)

> อยากได้ไอคอนอื่นเพิ่ม บอกผมได้ เดี๋ยวเพิ่มให้ในส่วน `ICONS`

**สี (`color`)** — ใส่ได้ตามนี้:
`var(--blue-500)` น้ำเงิน, `var(--blue-400)` ฟ้า, `var(--cyan-500)` ฟ้าคราม,
`var(--amber-500)` ส้ม, `var(--green-500)` เขียว, `var(--red-500)` แดง,
`var(--blue-700)` น้ำเงินเข้ม

---

## ⚠️ ถ้าแก้แล้วเว็บขาว/พัง

มักเกิดจากพิมพ์ผิดเล็กน้อย ตรวจ 3 จุดนี้:
1. ลืมจุลภาค `,` ท้ายบรรทัด หรือท้ายบล็อก `},`
2. เครื่องหมายคำพูดไม่ครบคู่ หรือใช้ `"` ซ้อนใน `"..."`
3. ลบ `{ }` หรือ `[ ]` ไม่ครบคู่

วิธีกู้: กด Ctrl+Z ย้อนกลับ หรือเก็บไฟล์สำรองไว้ก่อนแก้ทุกครั้ง

> แก้ไม่ได้จริงๆ ส่งไฟล์มาให้ผมช่วยดู หรือบอกว่าอยากเพิ่ม/แก้อะไร เดี๋ยวจัดให้ได้เลย
