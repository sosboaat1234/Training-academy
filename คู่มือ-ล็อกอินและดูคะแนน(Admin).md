# คู่มือ: ระบบล็อกอิน + ดูคะแนนผู้เรียน (Admin)

## 1) ระบบล็อกอินด้วยอีเมล @sitem.co.th

ตอนนี้เว็บมีหน้า Login แล้ว — ผู้เรียนต้องกรอกชื่อ + อีเมลที่ลงท้าย `@sitem.co.th`
ถึงจะเข้าใช้งานได้ และระบบจะจำการล็อกอินไว้ (รีเฟรชไม่ต้องกรอกใหม่) มีปุ่ม "ออก" ที่มุมขวาบน

**ข้อจำกัดที่ต้องเข้าใจ (สำคัญ):**
การล็อกอินนี้เป็น "ด่านกรองเบื้องต้น" ฝั่งหน้าเว็บ — เช็คแค่ว่าอีเมลลงท้าย @sitem.co.th
แต่ **ไม่ได้ยืนยันรหัสผ่านจริง** (เพราะเว็บฟรีไม่มี server) คนที่พิมพ์อีเมล @sitem.co.th มั่วก็เข้าได้
> ถ้าต้องการล็อกอินจริงแบบยืนยันตัวตน (กดปุ่ม "Sign in with Google" แล้วเข้าด้วยบัญชี
> @sitem.co.th จริงเท่านั้น) ทำได้ฟรีเหมือนกัน แต่ต้องตั้งค่า Google OAuth เพิ่ม 1 ครั้ง
> — บอกผมได้ถ้าอยากอัปเกรดเป็นแบบนี้

**เปลี่ยนโดเมน:** ถ้าใช้โดเมนอีเมลอื่น แก้ในไฟล์ `index.html` ที่บรรทัด
`const SITEM_DOMAIN = "@sitem.co.th";`

### เปิดใช้ "Sign in with Google" จริง (ยืนยันตัวตนจริง — ฟรี)
ตอนนี้เว็บรองรับปุ่ม Sign in with Google แล้ว แต่ต้องใส่ "OAuth Client ID" ของบริษัทก่อน
ปุ่มถึงจะโผล่ (ระหว่างนี้ใช้ช่องกรอกอีเมลไปก่อนได้) ทำครั้งเดียว:

1. ต้อง **อัปโหลดเว็บขึ้นโฮสต์ก่อน** (เช่น GitHub Pages/Netlify) แล้วจำ URL ไว้ เช่น
   `https://yourname.github.io` — ปุ่ม Google ทำงานเฉพาะบนเว็บที่ออนไลน์ (ไม่ทำงานตอนเปิดไฟล์จากเครื่อง)
2. เข้า https://console.cloud.google.com → สร้างโปรเจกต์ใหม่ (ฟรี)
3. เมนู **APIs & Services > OAuth consent screen** → เลือก Internal (ถ้าเป็น Workspace ของ SITEM) → กรอกชื่อแอป + อีเมล → บันทึก
4. เมนู **APIs & Services > Credentials > Create Credentials > OAuth client ID**
   - Application type: **Web application**
   - **Authorized JavaScript origins**: ใส่ URL เว็บของคุณ (เช่น `https://yourname.github.io`)
   - กด Create → จะได้ **Client ID** หน้าตาเช่น `1234567890-abc.apps.googleusercontent.com`
5. เปิด `index.html` หาบรรทัด `const GOOGLE_CLIENT_ID = "";`
   วาง Client ID ลงในเครื่องหมายคำพูด แล้วบันทึก + อัปโหลดทับ
6. เสร็จ! ปุ่ม "Sign in with Google" จะโผล่ และเข้าได้เฉพาะบัญชี @sitem.co.th จริงเท่านั้น

> โค้ดตั้งค่า `hd:"sitem.co.th"` ไว้แล้ว = Google จะแนะนำเฉพาะบัญชีบริษัท และระบบยังเช็คซ้ำว่า
> อีเมลลงท้าย @sitem.co.th ก่อนให้เข้า

## 2) ตั้งข้อสอบให้เก็บคะแนนอัตโนมัติ (Google Form → Google Sheet)

หัวใจของ "หน้า Admin ดูคะแนนทุกคน" คือ Google Form ที่ตั้งเป็นโหมด Quiz
เมื่อผู้เรียนทำเสร็จ คะแนนจะถูกบันทึกลง Google Sheet โดยอัตโนมัติ = นั่นคือหน้า Admin ของคุณ

**ตั้งค่าฟอร์มแต่ละบท (ทำครั้งเดียวต่อฟอร์ม):**

1. เปิดฟอร์มข้อสอบใน Google Forms
2. ไปที่ **Settings (⚙️) > Make this a quiz** เปิดใช้งาน → ตั้งเฉลยและคะแนนแต่ละข้อ
   (ระบบจะตรวจและให้คะแนนอัตโนมัติ)
3. ที่ Settings > Responses เปิด **Collect email addresses** (เก็บอีเมลผู้ตอบ)
   - ถ้าฟอร์มอยู่ใน Google Workspace ของบริษัท แนะนำตั้ง **Restrict to users in SITEM**
     เพื่อให้เฉพาะคน @sitem.co.th ตอบได้ และได้อีเมลจริงอัตโนมัติ
4. แนะนำเพิ่มคำถาม "ชื่อ-สกุล" และ "แผนก" ไว้ต้นฟอร์ม (เผื่ออยากดูเป็นชื่อ)
5. กด **Send > ไอคอนลิงก์ 🔗 > Copy** เอาลิงก์ (เช่น `https://forms.gle/xxxx`)
   ไปวางในช่อง `quizLink` ของบทนั้นใน `index.html`

> บท Road Blocker ผมใส่ลิงก์ `https://forms.gle/KzcGoX8dUNvXVEjM7` ให้แล้ว

## 3) ดูคะแนนผู้เรียนทุกคน (หน้า Admin)

1. ในฟอร์ม ไปแท็บ **Responses (คำตอบ)**
2. กดไอคอน **Google Sheets (สีเขียว)** เพื่อสร้าง/เปิดชีตที่ผูกกับฟอร์ม
3. ชีตนี้คือ "หน้า Admin" — แต่ละแถวคือผู้เรียน 1 คน มีคอลัมน์:
   เวลาส่ง (Timestamp), อีเมล, ชื่อ, และ **คะแนน (Score)** ที่ระบบตรวจให้
4. อยากดูสรุปสวยๆ: ในชีตทำ Pivot Table หรือกราฟได้ เช่น ใครยังไม่ส่ง / คะแนนเฉลี่ยรายบท

**ภาพรวมหลายบท:** ถ้ามีหลายฟอร์ม จะได้หลายชีต แยกกัน
- รวมง่ายๆ: สร้างชีตกลาง 1 อัน แล้วใช้ `IMPORTRANGE` ดึงคะแนนจากทุกฟอร์มมารวมในที่เดียว
- หรือบอกผม เดี๋ยวทำสูตร/หน้าสรุปรวมให้

## 4) เรื่อง "ดูคลิปจบหรือยัง"

- ปุ่ม **"ทำเครื่องหมายว่าเรียนจบ"** ในเว็บ จะอัปเดตวงแหวน % ของผู้เรียนคนนั้น
  แต่เก็บไว้ในเครื่องของเขาเอง (ไม่ส่งเข้าส่วนกลาง) — เหมาะให้ผู้เรียนติดตามตัวเอง
- ฝั่ง Admin ให้ยึด **"การส่งข้อสอบท้ายบท"** เป็นตัวชี้วัดว่าใครเรียนจบบทไหน
  (คนที่ส่งฟอร์มบทนั้น = ดูคลิป + ทำข้อสอบบทนั้นแล้ว) ซึ่งดูได้ครบใน Google Sheet
- ถ้าอยากเก็บ "ดูคลิปจบ" แยกเป็นข้อมูลส่วนกลางจริงๆ ต้องเพิ่ม Google Apps Script
  (ฟรี แต่ตั้งค่าเพิ่ม) — บอกได้ถ้าต้องการ
